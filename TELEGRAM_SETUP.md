# Настройка Telegram интеграции

Этот файл содержит инструкции по настройке интеграции с Telegram для получения заявок на примерку.

## Шаг 1: Создание Telegram бота

1. Откройте Telegram и найдите @BotFather
2. Напишите команду: `/newbot`
3. Введите имя для вашего бота (например: "KOSTUMERKA Fashion Bot")
4. Введите username для бота (должен заканчиваться на `bot`, например: "kostumerka_bot")
5. BotFather предоставит вам **токен** - сохраните его!
   Пример: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

## Шаг 2: Получение Chat ID

1. Найдите своего нового бота в Telegram или найдите @irina_fashionstyle
2. Напишите боту любое сообщение (например: `/start`)
3. Откройте в браузере:
   ```
   https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates
   ```
   Замените `<YOUR_BOT_TOKEN>` на ваш токен
4. В ответе найдите `"chat":{"id":` - это ваш Chat ID
   Пример: `123456789`

## Шаг 3: Настройка telegram-bot.js

1. Откройте файл `telegram-bot.js` в репозитории
2. Найдите секцию `TELEGRAM_CONFIG`:
   ```javascript
   const TELEGRAM_CONFIG = {
       botToken: 'YOUR_BOT_TOKEN_HERE',
       chatId: 'YOUR_CHAT_ID_HERE'
   };
   ```
3. Замените:
   - `YOUR_BOT_TOKEN_HERE` на ваш токен от BotFather
   - `YOUR_CHAT_ID_HERE` на ваш Chat ID

4. Сохраните файл и закоммитьте изменения

### Пример настройки:
```javascript
const TELEGRAM_CONFIG = {
    botToken: '1234567890:ABCdefGHIjklMNOpqrsTUVwxyz',
    chatId: '123456789'
};
```

## Шаг 4: Тестирование

1. Откройте ваш сайт: https://fashion-site-sigma.vercel.app
2. Найдите любой товар и нажмите "Записаться на примерку"
3. Заполните форму и отправьте
4. Проверьте Telegram - вы должны получить сообщение со всеми данными заявки

## Что будет в сообщении

Вы получите сообщение с такой информацией:
- Имя клиента
- Телефон
- Выбранные товары (название, категория, цена)
- Дата примерки
- Время примерки
- Комментарий (если есть)
- Время подачи заявки

## Устранение неполадок

### Сообщения не приходят
1. Проверьте правильность botToken и chatId
2. Убедитесь, что вы написали боту хотя бы одно сообщение
3. Откройте консоль разработчика (F12) и проверьте ошибки

### Ошибка "Unauthorized" или "Bad Request"
- Проверьте, что botToken скопирован полностью
- Уберите лишние пробелы и кавычки

### Форма отправляется, но сообщения нет
- Это нормально - форма работает даже если Telegram недоступен
- Проверьте консоль браузера на наличие ошибок
